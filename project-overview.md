# Project Overview

## Project Name
**Raghvendra Saini — Political Professional Portfolio & Admin Portal**

## Client
**Raghvendra Saini**
- BJP Leader & Coordinator, Social Media Monitoring Cell, Home Dept., UP Govt.
- DOB: 01 March 1992 | Fatehpur, Uttar Pradesh

## Purpose
A dual-purpose platform:
1. **Public Portfolio** — Presents the client's political identity, government work, achievements, and thoughts to voters, media, and party leadership.
2. **Admin Portal** — Allows the client to manage all content sections, upload documents, write blog posts ("My Thoughts"), track media coverage, and monitor engagement — without technical assistance.

## Target Audiences
| Audience | Goal |
|---|---|
| General Public / Voters | Learn about the leader, read his views |
| BJP Party Leadership | Evaluate political credentials & history |
| Government Officials | Review professional profile & documents |
| Media / Press | Access quotes, articles, press kit |
| Admin (Client) | Manage & update all content |

## Core Modules
1. Profile & Identity
2. Political & Organizational History
3. Government Scope of Work
4. My Thoughts (Blog / Opinion)
5. Media & Press Coverage
6. Community & Welfare Work
7. Documents Vault
8. Social Media Dashboard
9. Analytics
10. Settings & Privacy

## Tech Stack (Recommended)
- **Frontend**: Nextjs+ Tailwind CSS
- **Backend**: Nextjs and firabse
- **Auth**: Nextjs and firebase
- **Storage**: firstore Storage / Firebase for documents
- **Deployment**: Vercel or Netlify

## Languages
- Hindi (primary — for public-facing content)
- English (secondary — for professional/government sections)
